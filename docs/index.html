Prithvi

Container Security Scanner

Named after the Hindu Earth Goddess — protecting the foundation of your infrastructure

GitHub Get Started

Features

Dockerfile Analysis

7 built-in rules checking for root users, unpinned tags, hardcoded secrets, privileged ports, apt best practices, broad COPY statements, and missing healthchecks.

Image Scanning

Inspect container images for oversized layers, root execution, environment secrets, and known CVE vulnerabilities via local SQLite database.

Multiple Report Formats

Generate reports as rich CLI tables, JSON for CI/CD integration, or self-contained HTML reports with severity badges and remediation guidance.

CI/CD Ready

Non-zero exit codes on HIGH+ findings make Prithvi a drop-in security gate for your CI pipelines. Configure severity thresholds and rule exclusions.

Installation

git clone https://github.com/MukundaKatta/prithvi.git
cd prithvi
pip install -e ".[dev]"

Quick Start

# Scan a Dockerfile
prithvi scan dockerfile ./Dockerfile

# JSON output for CI pipelines
prithvi scan dockerfile ./Dockerfile --format json

# HTML report
prithvi scan dockerfile ./Dockerfile --format html --output report.html

# Only show HIGH and CRITICAL findings
prithvi scan dockerfile ./Dockerfile --severity-threshold HIGH

# Ignore specific rules
prithvi scan dockerfile ./Dockerfile --ignore DSC-004 --ignore DSC-007

# List all available rules
prithvi rules

Architecture

prithvi/
├── src/prithvi/
│   ├── cli.py              # Click CLI entry point
│   ├── models.py           # Finding, Severity, ScanResult
│   ├── config.py           # Configuration management
│   ├── dockerfile/         # Dockerfile analysis
│   │   ├── parser.py       # Instruction-level parser
│   │   ├── analyzer.py     # Analysis orchestrator
│   │   └── rules/          # 7 security rules (DSC-001 to DSC-007)
│   ├── image/              # Container image analysis
│   │   ├── inspector.py    # Tarball + daemon inspection
│   │   ├── layer.py        # Layer-level checks
│   │   ├── packages.py     # OS package extraction
│   │   └── analyzer.py     # Image scan orchestrator
│   ├── cve/                # CVE vulnerability database
│   │   ├── database.py     # SQLite storage
│   │   └── matcher.py      # Package-CVE matching
│   └── reporting/          # Output formatters
│       ├── json_reporter.py
│       ├── table_reporter.py
│       └── html_reporter.py
└── tests/                  # Comprehensive test suite
